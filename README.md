The purpose of thie repository is to provide a github repository associated with the USGS
ground-motion processing software [gmprocess](https://code.usgs.gov/ghsc/esi/groundmotion-processing).
Please use that repository as the source for the code and documentation.

A significant problem with that GitLab instance is that posting issues to repositories is
restricted to registered users, and registering a user is not immediate or automatic, requiring
an internal approval process. 

The purpose of this GitHub repository is to allow users to create issues associated with gmprocess,
without the added hurdles and delays of creating an account at [code.usgs.gov](code.usgs.gov). 

The developers can be reached at [gmprocess@usgs.gov](mailto:gmprocess@usgs.gov).
